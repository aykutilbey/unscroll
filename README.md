# 📵 unscroll

Sosyal medya bağımlılığını bilimsel kriterlerle ölçen ve kişiye özel detoks programları oluşturan web uygulaması.

🔗 **Canlı Site:** [unscroll-detox.vercel.app](https://unscroll-detox.vercel.app)

---

## Özellikler

- 9 soruluk bağımlılık testi
- Kişisel risk analizi (3 kategori: Güvenli / Uyarı / Kritik)
- Kişiye özel görev havuzu (kolay / orta / zor)
- Program PDF olarak indirilebilir
- Kullanıcı hesabı ile programa devam edilebilir
- Başarı hikayelerini paylaşma ve admin onay sistemi

## Teknolojiler

- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
- [Supabase](https://supabase.com)
- [Vercel](https://vercel.com)
